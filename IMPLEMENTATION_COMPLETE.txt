  ✓ Messages contextuels pour chaque étape
  ✓ Barre de progression animée
  ✓ Animation de chargement
  ✓ Compteur de données protégées

  ✓ Téléphones  → Masquage (+221 77****67)
  ✓ Wave/OM     → Masquage intelligent
  ✓ Banking     → Tokenisation (****-****-****-9012)
  ✓ CNI         → Chiffrement (CNI-12******23)
  ✓ NINEA       → Masquage (SN1****89)

Export de données :

  ✓ CSV (export universel)
  ✓ JSON (format structuré)
  ✓ Excel (XLSX)
  ✓ Rapport d'anonymisation (texte)

  ✓ Interface de saisie sécurisée

  ✓ Double saisie de confirmation
